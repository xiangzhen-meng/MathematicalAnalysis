# 多维空间的结构

## 概述

多维空间 R^n 是分析多元函数的基础。它同时具有**代数结构**（线性空间）和**几何结构**（距离/范数），这两种结构相互配合，使我们能够讨论极限、连续性和完备性。

## 代数结构：线性空间

### 向量的表示

采用上标表示坐标编号（张量分析的标准记法），避免与序列下标混淆：

$$
x = (x^1, x^2, \ldots, x^n) \in \mathbb{R}^n
$$

>[!note] 记号约定
>- 第 $i$ 个坐标的 $k$ 次方写作 $(x^i)^k$，避免与上标混淆
>- 同一字母附以下标表示序列中的点：$\{x_k\}$
>- 序列中第 $k$ 个点的第 $i$ 个坐标记为 $x_k^i$

## 几何结构：范数与距离

这里参考 [[距离与范数]] 
### 范数的等价性

>[!note] 定理
>在 $\mathbb{R}^n$ 中，任意两种范数 $\|\cdot\|_a$ 和 $\|\cdot\|_b$ 等价，即存在常数 $c_1, c_2 > 0$ 使得
>$$
c_1 \|x\|_a \leq \|x\|_b \leq c_2 \|x\|_a, \quad \forall x \in \mathbb{R}^n
>$$

**意义**：有限维赋范线性空间本质上等同于欧氏空间，收敛性、连续性等概念与范数选择无关

### 单位球的几何意义

范数的齐次性意味着单位球 $B(0,1) = \{x : \|x\| = 1\}$ 完全刻画了范数：

- **欧氏范数**：单位球为普通球面
- **∞-范数**：单位球为边长为 2 的立方体 $\{x : |x^i| \leq 1, \forall i\}$
- **1-范数**：单位球为八面体 $\{x : \sum |x^i| \leq 1\}$

## 拓扑结构：开集与闭集

### 开集与闭集

>[!note] 开集
>集合 $G \subset \mathbb{R}^n$ 称为开集，若 $\forall x \in G$，$\exists \eta > 0$ 使得 $U(x, \eta) \subset G$。

等价地，$G$ 是开集当且仅当 $G = \text{int}(G)$（$G$ 的每一点都是内点）。

>[!note] 闭集
>集合 $F \subset \mathbb{R}^n$ 称为闭集，若其中任何收敛点列的极限仍属于 $F$。

等价地，$F$ 是闭集当且仅当 $F^c$ 是开集

注意，**聚点并不一定处于集合内**。聚点在集合内，当且仅当 **集合是闭包**

### 内点、外点与边界点

对于集合 $E \subset \mathbb{R}^n$：

| 类型      | 定义             | 条件                                                                                                 |
| ------- | -------------- | -------------------------------------------------------------------------------------------------- |
| **内点**  | $a$ 是 $E$ 的内点  | $\exists \delta > 0$，$U(a, \delta) \subset E$                                                      |
| **外点**  | $a$ 是 $E$ 的外点  | $\exists \delta > 0$，$U(a, \delta) \subset E^c$                                                    |
| **边界点** | $a$ 是 $E$ 的边界点 | $\forall \delta > 0$，$U(a, \delta) \cap E \neq \emptyset$ 且 $U(a, \delta) \cap E^c \neq \emptyset$ |
| **聚点**  | $a$ 是 $E$ 的聚点  | $\forall \delta > 0$，$\mathring{U}(a, \delta) \cap E \neq \emptyset$                               |
|         |                |                                                                                                    |

**记号**
- 内部：$\text{int}(E)$ 或 $E^\circ$（所有内点）
- 外部：$\text{ext}(E)$（所有外点）
- 边界：$\partial E$ 或 $\text{bdry}(E)$（所有边界点）
- 闭包：$\bar{E} = E \cup \partial E = \text{int}(E) \cup \partial E$（包含 $E$ 的最小闭集）

## 收敛点列

### 收敛的定义

>[!note] 点列收敛
>设 $\{x_k\}$ 是 $\mathbb{R}^n$ 中的点列，$a \in \mathbb{R}^n$。若
>$$
\lim_{k \to \infty} \|x_k - a\| = 0
>$$
>则称 $\{x_k\}$ **收敛**于 $a$，记为 $\lim_{k \to \infty} x_k = a$。

**ε-N 语言**：
$$
\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall k > N : \|x_k - a\| < \varepsilon
$$

**几何解释**：点列 $\{x_k\}$ 从第 $N$ 项以后全部落入 $a$ 的任意小邻域 $U(a, \varepsilon)$ 中。

### 按坐标收敛定理

>[!note] 定理
>对于 $\mathbb{R}^n$ 中的点列 $\{x_k\}$ 和点 $a$，以下等价：
>$$
\lim_{k \to \infty} x_k = a \quad \Leftrightarrow \quad \lim_{k \to \infty} x_k^i = a^i, \quad i = 1, 2, \ldots, n
>$$

**证明**：利用范数不等式
$$
\max_{1 \leq i \leq n} |x_k^i - a^i| \leq \|x_k - a\|_2 \leq \sum_{i=1}^n |x_k^i - a^i|
$$

左端趋于 0 意味着每坐标都趋于对应坐标；右端趋于 0 可由各坐标收敛推出。

**意义**：高维点列的收敛性归结为各坐标分量的收敛性，将问题转化为一维情形。

## 柯西点列与完备性

### 柯西点列

>[!note] 柯西点列（基本列）
>点列 $\{x_k\}$ 称为柯西点列，若
>$$
\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall m, k > N : \|x_m - x_k\| < \varepsilon
>$$

**几何意义**：点列中的点随着序号增大而彼此任意接近。

### 完备性

>[!note] 定理
>**收敛列必为柯西列**。

**证明**：设 $\lim x_k = a$，则 $\forall \varepsilon > 0$，$\exists N$，$\forall k > N$ 有 $\|x_k - a\| < \varepsilon/2$。于是对 $m, k > N$：
$$
\|x_m - x_k\| \leq \|x_m - a\| + \|a - x_k\| < \varepsilon
$$

>[!question] 柯西列是否必收敛？
在一维空间 $\mathbb{R}$ 中，柯西列必收敛（实数系的完备性）。但在一般的距离空间中，柯西列未必收敛。

**反例**：开区间 $(0, 1)$ 中的序列 $x_k = 1/k$ 是柯西列，但极限 $0 \notin (0, 1)$，故不收敛。

>[!note] 完备空间
>距离空间 $(X, d)$ 称为完备的，若其中的每个柯西点列都在 $X$ 中收敛。

**定理**：$\mathbb{R}^n$（赋予欧氏距离或任意等价范数）是完备的距离空间。

**意义**：完备性保证了极限运算的封闭性，是分析学中许多定理成立的基础（如压缩映射原理）
