# 积分中值定理

## 概述

积分中值定理揭示了连续函数在闭区间上的积分与其函数值之间的关系，表明积分值可以表示为某点函数值与区间长度的乘积。

目前学到的是积分第一中值定理 & 积分第二中值定理

## 定理

### 基本形式

>[!note] 积分第一中值定理（基本形式）
>设 $f \in C[a,b]$，则存在 $c \in [a,b]$，使得
>$$\int_a^b f(x)\,dx = f(c)(b-a)$$
>
>其中 $f(c) = \frac{1}{b-a}\int_a^b f(x)\,dx$ 称为 $f$ 在 $[a,b]$ 上的**积分平均值**。

### 加权形式

>[!note] 定理 7.12（积分第一中值定理——加权形式）
>设 $f, g$ 在 $[a,b]$ 上可积，且满足
>$$m \leq f(x) \leq M, \quad g(x) \geq 0, \quad \forall x \in [a,b]$$
>
>则有不等式
>$$m\int_a^b g(x)\,dx \leq \int_a^b f(x)g(x)\,dx \leq M\int_a^b g(x)\,dx$$
>
>特别地，若 $f \in C[a,b]$ 且 $\int_a^b g(x)\,dx \neq 0$，则存在 $c \in [a,b]$，使得
>$$\int_a^b f(x)g(x)\,dx = f(c)\int_a^b g(x)\,dx$$

**注**：若 $g(x) \leq 0$，定理的第一个结论中的不等式反向，但第二个结论仍成立。由此可以建立一个对于加权形式的直观：如果不要求权值不变号，则将 $g$ 用 $-g$ 代，得到反号形式，于是三者全部相等，这是不可能的

**加权形式的意义**：
- 相当于以 $g(x)$ 为权重的加权平均
- 当 $g(x) \equiv 1$ 时退化为基本形式

### 积分第二中值定理

积分第二中值定理研究的是**积分限**的中值选取问题，与第一中值定理研究被积函数的中值不同。

>[!note] 定理 7.13（积分第二中值定理——特殊形式）
>设 $f$ 在 $[a,b]$ 上单调下降且非负，$g$ 在 $[a,b]$ 上可积，则存在 $c \in [a,b]$，使得
>$$\int_a^b f(x)g(x)\,dx = f(a)\int_a^c g(x)\,dx$$

>[!note] 定理 7.14（积分第二中值定理——一般形式）
>设 $f$ 在 $[a,b]$ 上单调，$g$ 在 $[a,b]$ 上可积，则存在 $c \in [a,b]$，使得
>$$\int_a^b f(x)g(x)\,dx = f(a)\int_a^c g(x)\,dx + f(b)\int_c^b g(x)\,dx$$

**定理 7.14 的特例**：
1. 当 $f \equiv 1$ 时：$\int_a^b g(x)\,dx = \int_a^c g(x)\,dx + \int_c^b g(x)\,dx$（恒等式）
2. 当 $f(a) = 1, f(b) = 0$ 时：$\int_a^b f(x)g(x)\,dx = \int_a^c g(x)\,dx$
3. 当 $f = g$ 且单调时：$\int_a^b f^2(x)\,dx = f(a)\int_a^c f(x)\,dx + f(b)\int_c^b f(x)\,dx$

**重要说明**：$f$ 的单调性是必需的。若 $f(a) = f(b) = 0$（如 $f(x) = \sin x, a=0, b=\pi$），对于一般的 $g(x)$，不一定有定理成立。

**理解**：
1. 对 7.13 特殊情形：考虑变限积分。由于 $f$ 单调递减，如果在最左端，必然有 $f(a)$ 是最大值。另一方面，考虑 $g$ 的积分，不妨让 $g$ 定号，于是往右积分肯定越积越多，形象地理解为 $g$ 是递增的。这个定理就是说明，一个递减的权重和一个被积函数，他们的变化是**连续的**。固定权重的一端，和一个积分限，一定能够找到对应的某个介值。
2. 不妨可以假设 $f,g$ 是足够光滑的，可以比较容易的推出结论

### Abel 和差变换公式

>[!note] 引理 7.6（Abel 和差变换公式）
>设 $\{a_i\}_{i=0}^n$ 和 $\{\beta_i\}_{i=0}^n$ 是两组实数，则
>$$\sum_{i=1}^n a_{i-1}(\beta_i - \beta_{i-1}) = \sum_{i=1}^n (a_i - a_{i-1})\beta_i + a_n\beta_n - a_0\beta_0$$
>
>或写成
>$$\sum_{i=1}^n a_{i-1}\Delta\beta_i = -\sum_{i=1}^n \beta_i\Delta a_i + a_n\beta_n - a_0\beta_0$$

**关键洞察**：Abel 和差变换公式是**离散意义下的分部积分公式**。

写一下就发现跟分布积分非常像：
$$
\int_1^n \alpha d\beta + \int_1^n \beta d\alpha = \alpha(n)\beta(n) - \alpha(1)\beta(1)
$$

## 证明

### 证明一：积分第一中值定理（基本形式）

**核心思路**：利用积分保号性和介值定理。

**步骤**：

1. **建立不等式**：由 $f$ 在 $[a,b]$ 上的连续性，设 $m = \min_{x\in[a,b]} f(x)$，$M = \max_{x\in[a,b]} f(x)$，则
   $$m \leq f(x) \leq M, \quad \forall x \in [a,b]$$

2. **积分保号性**：对上述不等式积分，得
   $$m(b-a) \leq \int_a^b f(x)\,dx \leq M(b-a)$$

3. **构造中间值**：令 $\mu = \frac{1}{b-a}\int_a^b f(x)\,dx$，则 $m \leq \mu \leq M$

4. **应用介值定理**：由 $f \in C[a,b]$，根据介值定理，存在 $c \in [a,b]$，使得 $f(c) = \mu$

5. **结论**：
   $$\int_a^b f(x)\,dx = f(c)(b-a)$$

### 证明二：加权形式

**核心思路**：将基本形式的证明推广到加权情形。

**步骤**：

1. **建立加权不等式**：由 $m \leq f(x) \leq M$ 和 $g(x) \geq 0$，得
   $$mg(x) \leq f(x)g(x) \leq Mg(x)$$

2. **积分得**：
   $$m\int_a^b g(x)\,dx \leq \int_a^b f(x)g(x)\,dx \leq M\int_a^b g(x)\,dx$$

3. **若 $\int_a^b g(x)\,dx = 0$**：由 $g(x) \geq 0$ 知 $g(x) = 0$ a.e.，等式显然成立

4. **若 $\int_a^b g(x)\,dx \neq 0$**：令
   $$\mu = \frac{\int_a^b f(x)g(x)\,dx}{\int_a^b g(x)\,dx}$$
   则 $m \leq \mu \leq M$，由介值定理得证。

### 证明三：Abel 和差变换公式

**核心思路**：重新组合求和项。

**证明**：注意到

$$\sum_{i=1}^n a_i\beta_i = \beta_n a_n + \sum_{i=1}^{n-1}(a_i - a_{i+1})\beta_i$$

整理得

$$\sum_{i=1}^n a_i\beta_i = \sum_{i=1}^n(a_i - a_{i-1})\beta_i + a_0\beta_0$$

移项即得所求公式。

### 证明四：积分第二中值定理（定理 7.13）

**核心思路**：利用分部积分和第一中值定理。

#### 简单情形：$f \in C^1[a,b], g \in C[a,b]$ 

1. **定义辅助函数**：令 $G(x) = \int_a^x g(t)\,dt$，则 $G \in C[a,b]$，$G(a) = 0$，且 $G'(x) = g(x)$

2. **分部积分**：
   $$\int_a^b f(x)g(x)\,dx = \int_a^b f(x)\,dG(x) = f(x)G(x)\big|_a^b - \int_a^b G(x)f'(x)\,dx$$

3. **利用 $f$ 单调下降**：由 $f'(x) \leq 0$，得
   $$\int_a^b f(x)g(x)\,dx = f(b)G(b) + \int_a^b (-f'(x))G(x)\,dx$$

4. **估计积分**：设 $m = \min_{x\in[a,b]} G(x)$，$M = \max_{x\in[a,b]} G(x)$，则
   $$\int_a^b (-f'(x))G(x)\,dx \geq m\int_a^b (-f'(x))\,dx = m(f(a) - f(b))$$

5. **综合得**：
   $$\int_a^b f(x)g(x)\,dx \geq f(b)G(b) + m(f(a) - f(b)) = f(a)m + f(b)(G(b) - m) \geq f(a)m$$

6. **同理可得上界**：
   $$\int_a^b f(x)g(x)\,dx \leq f(a)M$$

7. **应用介值定理**：由 $G$ 的连续性，存在 $c \in [a,b]$，使得
   $$\int_a^b f(x)g(x)\,dx = f(a)G(c) = f(a)\int_a^c g(x)\,dx$$

#### 一般情形（离散版本）的详细证明

当 $f$ 只是单调下降（不一定可导）时，我们需要使用 Abel 和差变换公式。

**步骤**：

1. **定义辅助函数**：令 $G(x) = \int_a^x g(t)\,dt$，则 $G \in C[a,b]$，$G(a) = 0$

2. **构造黎曼和**：对区间 $[a,b]$ 作分割 $P = \{a = x_0 < x_1 < \cdots < x_N = b\}$，考虑
   $$\sigma(fg, P, \xi) = \sum_{i=1}^N f(\xi_i)g(\xi_i)(x_i - x_{i-1})$$
   其中 $\xi_i \in [x_{i-1}, x_i]$

3. **使用积分中值定理**：对每个小区间 $[x_{i-1}, x_i]$，存在 $\eta_i \in [x_{i-1}, x_i]$，使得
   $$\int_{x_{i-1}}^{x_i} g(t)\,dt = g(\eta_i)(x_i - x_{i-1})$$
   即 $G(x_i) - G(x_{i-1}) = g(\eta_i)\Delta x_i$

4. **重写黎曼和**：取 $\xi_i = x_{i-1}$（左端点），则
   $$\sum_{i=1}^N f(x_{i-1})g(\eta_i)\Delta x_i = \sum_{i=1}^N f(x_{i-1})(G(x_i) - G(x_{i-1}))$$

5. **应用 Abel 和差变换**：令 $a_i = f(x_i)$，$\beta_i = G(x_i)$，则
   $$\sum_{i=1}^N f(x_{i-1})(G(x_i) - G(x_{i-1})) = \sum_{i=1}^N (f(x_{i-1}) - f(x_i))G(x_i) + f(x_N)G(x_N) - f(x_0)G(x_0)$$

6. **利用边界条件**：由于 $G(x_0) = G(a) = 0$，$x_N = b$，$x_0 = a$，得
   $$= \sum_{i=1}^N (f(x_{i-1}) - f(x_i))G(x_i) + f(b)G(b)$$

7. **利用 $f$ 单调性**：由 $f$ 单调下降，$f(x_{i-1}) - f(x_i) \geq 0$，设 $m = \min_{x\in[a,b]} G(x)$，$M = \max_{x\in[a,b]} G(x)$，则
   $$\sum_{i=1}^N (f(x_{i-1}) - f(x_i))m \leq \sum_{i=1}^N (f(x_{i-1}) - f(x_i))G(x_i) \leq \sum_{i=1}^N (f(x_{i-1}) - f(x_i))M$$

8. **求和化简**：
   $$\sum_{i=1}^N (f(x_{i-1}) - f(x_i)) = f(a) - f(b)$$
   因此
   $$(f(a) - f(b))m \leq \sum_{i=1}^N (f(x_{i-1}) - f(x_i))G(x_i) \leq (f(a) - f(b))M$$

9. **综合估计**：
   $$(f(a) - f(b))m + f(b)G(b) \leq \sigma(fg, P, \xi) \leq (f(a) - f(b))M + f(b)G(b)$$

10. **简化**：注意 $G(b) = \int_a^b g(t)\,dt$，且 $f(b) \geq 0$（非负假设），可得
    $$f(a)m \leq \sigma(fg, P, \xi) \leq f(a)M$$

11. **取极限**：当 $|P| \to 0$ 时，黎曼和收敛到积分，故
    $$f(a)m \leq \int_a^b f(x)g(x)\,dx \leq f(a)M$$

12. **应用介值定理**：由 $G$ 的连续性，存在 $c \in [a,b]$，使得
    $$\int_a^b f(x)g(x)\,dx = f(a)G(c) = f(a)\int_a^c g(x)\,dx$$

### 证明五：积分第二中值定理（定理 7.14）

**核心思路**：将一般单调函数转化为非负单调下降函数。

**步骤**：

1. **标准化**：令 $\tilde{f}(x) = f(x) - f(b)$，则 $\tilde{f}$ 单调下降且 $\tilde{f}(b) = 0$

2. **应用定理 7.13**：存在 $c \in [a,b]$，使得
   $$\int_a^b \tilde{f}(x)g(x)\,dx = \tilde{f}(a)\int_a^c g(x)\,dx = (f(a) - f(b))\int_a^c g(x)\,dx$$

3. **还原**：
   $$\int_a^b f(x)g(x)\,dx - f(b)\int_a^b g(x)\,dx = (f(a) - f(b))\int_a^c g(x)\,dx$$

4. **整理得**：
   $$\int_a^b f(x)g(x)\,dx = f(a)\int_a^c g(x)\,dx + f(b)\left(\int_a^b g(x)\,dx - \int_a^c g(x)\,dx\right)$$
   $$= f(a)\int_a^c g(x)\,dx + f(b)\int_c^b g(x)\,dx$$

5. **单调递增情形**：若 $f$ 单调递增，则 $-f$ 单调递减，上述结论乘以 $-1$ 即得。

## 理解

**两个中值定理的区别**：

- **第一中值定理**：$\int_a^b f(x)g(x)\,dx = f(c)\int_a^b g(x)\,dx$ —— 被积函数 $f$ 取中值
- **第二中值定理**：$\int_a^b f(x)g(x)\,dx = f(a)\int_a^c g(x)\,dx + f(b)\int_c^b g(x)\,dx$ —— 积分限取中值
- 第二中值定理不要求 $g$ 定号，但要求 $f$ 单调
